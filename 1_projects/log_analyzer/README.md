# Log Analyzer

A Python security tool that reads system log files, detects threats, and generates automated reports.

## What It Does

- Reads any system log file
- Categorizes events by severity: CRITICAL, ERROR, and WARNING
- Instantly flags dangerous activity
- Saves a full report to a text file automatically

## How To Run

bash
python3 loganalyzer.py

### Output

Prints a formatted report to the terminal and saves it as 'log_report.txt'

### Example Threats detected

- Brute force login attempts
- Unauthorized access attempts
- SQL injection attempts
- External root login attempts

### Built With

- Python 3
- Core modules: open(), readlines(), file write
