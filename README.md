# DevRehannaLP - Active Projects

A collection of real-world tools built while training as a DevSecOps Engineer.  Every project here was built from scratch, typed by hand, debugged and deployed.

## Projects

### 🔍 Log Analyzer
A Python security tool that reads system log files, detects threats, and generates automated reports.
- Detects CRITICAL, ERROR, and WARNING events
- Flags brute force attacks, SQL injection, unauthorized access
- Auto-generates a saved report file
- **Tech:** Python 3

###  🔐 Password Strength Checker
A Python tool that analyzes password strength and scores it based on length, complexity, and character variety.
- Hidden password input using getpass module
- Scoring system with feedback
- Multiple version developed and improved
- **Tech:** Python 3

### 🗃️ File Organizer
A Python automation tool that sorts files into catergorized folders automatically by file extension.
- Uses os and shutil modules
- Saves hours of manual file sorting
- **Tech:** Python 3
