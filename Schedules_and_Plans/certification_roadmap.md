# Certification Roadmap  
*A long‑term plan for building cybersecurity knowledge, skills, and credentials.*

---

## **Purpose**
This roadmap outlines the certifications to pursue as you grow from foundational cybersecurity knowledge into more advanced SOC, blue‑team, and security analyst roles.  
It provides structure, clarity, and a realistic progression path.

---

# **Phase 1 — Foundations (0–6 Months)**

## **Google Cybersecurity Certificate**
**Status:** In progress / Completed  
**Focus Areas:**
- Security fundamentals  
- Linux basics  
- Networking  
- SIEM analysis  
- Incident response  
- Risk and controls  

**Why it matters:**  
Builds the essential baseline knowledge needed for SOC and entry‑level roles.

---

## **CompTIA Security+ (Optional but Valuable)**
**Focus Areas:**
- Threats, attacks, and vulnerabilities  
- Architecture and design  
- Identity and access management  
- Risk management  
- Cryptography  

**Why it matters:**  
Industry‑recognized baseline certification for cybersecurity roles.

---

# **Phase 2 — SOC & Blue Team Skills (6–12 Months)**

## **CompTIA CySA+ (Cybersecurity Analyst)**
**Focus Areas:**
- Log analysis  
- Threat detection  
- Vulnerability management  
- Incident response  
- Behavioral analytics  

**Why it matters:**  
Aligns directly with SOC analyst responsibilities.

---

## **Splunk Core User / Power User (Optional)**
**Focus Areas:**
- Searching and filtering logs  
- Creating dashboards  
- Understanding SIEM workflows  

**Why it matters:**  
Splunk is widely used in SOC environments.

---

# **Phase 3 — Intermediate Specialization (12–24 Months)**

## **eJPT (Junior Penetration Tester)**
**Focus Areas:**
- Networking  
- Enumeration  
- Exploitation basics  
- Web vulnerabilities  

**Why it matters:**  
Strengthens understanding of attacker behavior — valuable for blue‑team analysts.

---

## **Blue Team Level 1 (BTL1)**
**Focus Areas:**
- SOC operations  
- Threat hunting  
- Detection engineering  
- Log analysis  
- Incident response  

**Why it matters:**  
Highly respected hands‑on blue‑team certification.

---

# **Phase 4 — Advanced Pathways (2+ Years)**

## **eCIR (Incident Responder)**
**Focus Areas:**
- Forensics  
- Incident handling  
- Malware triage  
- Threat intelligence  

---

## **Azure or AWS Security Specialty**
**Focus Areas:**
- Cloud security  
- Identity and access management  
- Logging and monitoring  
- Cloud incident response  

---

# **Long‑Term Optional Goals**
These are advanced certifications you may pursue later:

- **CISSP** — management‑level security certification  
- **GSEC / GCIA / GCIH** — advanced SANS certifications  
- **OSCP** — advanced penetration testing
