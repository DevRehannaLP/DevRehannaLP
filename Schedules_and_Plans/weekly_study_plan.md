# Weekly Study Plan  
*A structured weekly routine for consistent cybersecurity learning.*

---

## **Purpose**
This weekly plan helps maintain steady progress across courses, labs, projects, and career preparation. It balances learning, hands‑on practice, and review.

---

## **Weekly Schedule**

### **Monday — Google Cybersecurity Certificate**
- Watch course videos  
- Take notes  
- Complete module quizzes  
- Update Google_Certificate_Notes folder  

### **Tuesday — TryHackMe Labs**
- Complete 1–2 rooms  
- Focus on Linux, SOC, or Blue Team paths  
- Document findings in TryHackMe-Labs folder  

### **Wednesday — SOC Tools Practice**
- Practice with Splunk, Wireshark, Zeek, or Suricata  
- Review logs, filters, and detection techniques  
- Update SOC-Tools notes  

### **Thursday — Python & Automation**
- Work on Python scripts  
- Improve automation tools  
- Add updates to Python-Scripts folder  

### **Friday — Projects & Portfolio**
- Work on SOC, malware, or network investigation projects  
- Update project README files  
- Commit changes to GitHub  

### **Saturday — Review & Reinforcement**
- Review notes from the week  
- Revisit difficult topics  
- Update study materials  

### **Sunday — Rest or Light Review**
- Optional: light reading, videos, or recap  
- No heavy study required  

---

## **Tips for Success**
- Stay consistent  
- Keep notes organized  
- Commit to GitHub regularly  
- Track progress weekly  
- Adjust the schedule as needed
